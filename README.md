# Unsplash Background Changer

PowerShell скрипт для автоматической установки случайных обоев с Unsplash.

## Возможности

- Загрузка случайных изображений с Unsplash API
- Автоматическая установка обоев рабочего стола
- Настраиваемые категории и размеры изображений
- Логирование операций
- Конфигурация через JSON файл

## Требования

- Windows 10/11
- PowerShell 5.1 или выше
- Доступ к интернету
- API ключ Unsplash (опционально)

## Установка

1. Скачайте все файлы проекта в одну папку
2. Получите API ключ на [Unsplash Developers](https://unsplash.com/developers)
3. Укажите ключ в файле `config.json`

## Использование

### Базовое использование
```powershell
.\Unsplash-BG.ps1
```

### С параметрами
```powershell
.\Unsplash-BG.ps1 -Category "city" -Width 2560 -Height 1440
```

### Параметры

- `-Category` - категория изображений (по умолчанию: "nature")
- `-Width` - ширина изображения (по умолчанию: 1920)
- `-Height` - высота изображения (по умолчанию: 1080)

## Конфигурация

Отредактируйте файл `config.json` для настройки:

- `unsplash.accessKey` - API ключ Unsplash
- `unsplash.defaultCategory` - категория по умолчанию
- `download.tempPath` - папка для временных файлов
- `wallpaper.style` - стиль обоев (fill, fit, stretch, center, tile)

## Автозапуск

Для автоматической смены обоев добавьте скрипт в планировщик задач Windows:

1. Откройте "Планировщик заданий"
2. Создайте новую задачу
3. Укажите путь к PowerShell и скрипту
4. Настройте расписание (например, каждые 30 минут)

## Логи

Скрипт создает файл `unsplash-bg.log` с информацией о работе.
