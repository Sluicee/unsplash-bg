# Установка Unsplash Background Changer

## Быстрый старт

1. **Получите API ключ Unsplash:**
   - Перейдите на https://unsplash.com/developers
   - Создайте новое приложение
   - Скопируйте Access Key

2. **Настройте приложение:**
   - Запустите Config.exe
   - Выберите "1. API Settings"
   - Введите ваш API ключ
   - Настройте другие параметры по желанию
   - Сохраните конфигурацию

3. **Запустите смену обоев:**
   - Запустите Unsplash-BG.exe для ручной смены
   - Или настройте автозапуск через Config.exe

## Файлы

- Unsplash-BG.exe - основной скрипт смены обоев
- Config.exe - конфигуратор
- config.json - файл конфигурации
- README.md - подробная документация

## Автозапуск

Для автоматической смены обоев:
1. Запустите Config.exe
2. Выберите "4. Auto-change Settings"
3. Включите автозамену и настройте интервал
4. Выберите "5. Task Scheduler Setup"
5. Создайте задачу планировщика

## Поддержка

При возникновении проблем проверьте:
- Настроен ли API ключ
- Есть ли доступ к интернету
- Запущено ли приложение от имени администратора
